NSDL（NES Sound Driver Library）

トップ＼ 使い方＼ 音源ドライバー＼ 関数リファレンス＼

nsd_play_dpcm()

定義（Define）

        void    __fastcall__    nsd_set_dpcm(void* dpcm );

引数（Inputs）

void* dpcm … ⊿PCM制御用構造体のポインターを指定してください。
Please set the pointer of delta PCM information.

戻り値（Outputs）

無し（no output）

内容（Contents）

引数void* dpcmで指定された⊿PCM制御用構造体をロードします。
⊿PCM制御用構造体には、各音階毎に波形データの配置アドレス、サイズ、再生周波数、モード等の情報があります。

Sound driver will set the delta PCM information that was set to void* dpcm.

注意（Notes）

⊿PCMを用いた曲データを再生する際は、nsd_play_bgm()関数を呼び出す前に必ず呼び出してください。
