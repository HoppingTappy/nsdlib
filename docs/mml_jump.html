NSDL（NES Sound Driver Library）

トップ＼ 使い方＼ MMLコンパイラ＼ MML文法＼ MMLシーケンス＼

MML reference: Jump

定義（Define）

        Jn

引数（Inputs）

n

0
無視を解除します。
1
以後の音符、休符、ポルタメント、サブルーチンコール、効果音呼出、次の音符のみの相対オクターブコマンドを無視します。

内容（Contents）

以後の音符、休符、ポルタメント、サブルーチンコール、効果音呼出、次の音符のみの相対オクターブコマンドを無視することにより、 その他のコマンドを反映しながら、曲を J1 の地点から、 J0 の地点へ強制的に飛ばすことが可能になります。

注意（Notes）

その他のコマンドは反映されますが、サブルーチン内のコマンドは反映されない旨、注意してください。

例（Sample）

bgm(0){

    TR1,2,3     J1          //ここから、演奏を飛ばします。

    t120

    TR1 l4  o5  c1<
    TR2 l4  o4  e1
    TR3 l4  o3  c1

    TR1,2,3     J0          //ここから、演奏が開始されますが、コマンドは反映されています。

    TR1     b1	
    TR2     d1	
    TR3     g1	

}
