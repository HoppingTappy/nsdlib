MML reference: FDS modulator frequency

定義（Define）

        @FFfrequency

引数（Inputs）

frequency

モジュレーターの周波数を0（停止）及び、1（低周波）～4095（高周波）の範囲で設定してください。（Default: 不定）
発振周波数は、 ( 1789772.5 ÷ ( 65536 × 64 ) ) × frequency [Hz] となります。

内容（Contents）

モジュレーターの周期を指定します。

注意（Notes）

例（Sample）

    @FF16     /* FDS modulator frequency */
