NSDL（NES Sound Driver Library）

トップ＼ 使い方＼ 音源ドライバー＼ 関数リファレンス＼

nsd_save()

定義（Define）

        void    __fastcall__    nsd_save(void* buff );

引数（Inputs）

void* buff … 制御用構造体の内容を転送するポインタを指定してください。
転送先のメモリ領域は、 nsd_work_zp構造体及び、 nsd_work構造体の合計サイズが必要です。

戻り値（Outputs）

無し（no output）

内容（Contents）

現在の演奏状態を、指定のポインタに保存します。
保存した内容は、nsd_load()関数で再び元に戻すことができます。

注意（Notes）

本関数を呼出すと、指定ポインタから制御用構造体へ転送するために、ＢＧＭ，効果音共に発音を停止します。
ＢＧＭは、nsd_resume_bgm()関数にて再生を再開する事が可能です。

転送先のメモリには、nsd_work_zp構造体及び、 nsd_work構造体の合計サイズが必要です。
本体内のＲＡＭを使用した場合は圧迫することが予想され、保存先はバッテリーバックアップＲＡＭ等の利用をお勧めします。
