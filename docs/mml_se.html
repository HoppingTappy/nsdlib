MML reference: Call SE

定義（Define）

        SEn

引数（Inputs）

n

呼び出す効果音（SE）の番号です。

内容（Contents）

n番の効果音（SE）を呼び出します。

注意（Notes）

効果音で使用しているトラック（APUパルス波ch2及び、APUノイズ）の発音は一時的に停止します。
音源ドライバーの内部的には、nsd_play_se(void *se)関数を呼び出しているだけです。

例（Sample）

/* Block of BGM[0] */
bgm(0){
	TR1	Ev11	v15	o4		l1	SE0	cdef	SE1	gab`c
	TR2	Ev11	v8	o4	r8.	l1		cdef		gab`c
	TR4	Ev10	v12	o6		l16		[32	cc"gc	]
}

/* Block of SE[0] */
SE(0){
	TR1	t255	v11	Ev1	Em101	E@201	l4	c.d.e f.g.a 
}

/* Block of SE[1] */
SE(1){
	TR2	t255	v15	Ev10	[8	c4	]
}
