NSDL（NES Sound Driver Library）

トップ＼ 使い方＼ 音源ドライバー＼ 関数リファレンス＼

nsd_init()

定義（Define）

        void    __fastcall__    nsd_init(void );

引数（Inputs）

無し（no input）

戻り値（Outputs）

無し（no output）

内容（Contents）

サウンドドライバーを初期化します。
ユーザーアプリケーションの起動時に、この関数を呼び出して下さい。

注意（Notes）

nsd_main()関数を呼び出す前に必ず呼び出してください。
本関数では、VBlank割り込み（NMI）の操作は行いません。
タイマー制御にVBlank割り込み（NMI）を用いる場合は、本関数を呼び出し後に、 ユーザーアプリケーション側にてVBlank割り込みを`有効'にPPUを設定してください。
