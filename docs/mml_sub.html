MML reference: Subroutine

定義（Define）

        Sn

引数（Inputs）

n

呼び出すサブルーチンの番号です。

内容（Contents）

n番のサブルーチンを呼び出します。
何度も使用するシーケンスはサブルーチン化することにより、曲データのサイズを小さくすることができます。
同じＭＭＬファイル内であれば、異なるＢＧＭやＳＥからでも、同じサブルーチンを使用できます。

注意（Notes）

サブルーチン内からは、サブルーチンの呼び出しはできません。
曲データのサイズを小さくするためには、 リピート(A)コマンド、 リピート(B)コマンド及び、このサブルーチンコマンドで、うまく最適化してください。

例（Sample）

/* Block of subroutine */
Sub(0){
    l4 o4 cdefgab>c
} 

/* Block of BGM[0] */
BGM(0){
    TR1     t120 S0
}

/* Block of BGM[1] */
BGM(1){
    TR1     t80 S0
}
