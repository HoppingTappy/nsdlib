MML reference: Echo Buffer Control

定義（Define）

        ECcontrol(Accidentals)(octave)

引数（Inputs）

control

control contents
- 疑似エコーのバッファを、全てリセットします。
r 疑似エコーのバッファに休符を書き込みます。
a 疑似エコーのバッファにラを書き込みます。
b 疑似エコーのバッファにシを書き込みます。
c 疑似エコーのバッファにドを書き込みます。
d 疑似エコーのバッファにレを書き込みます。
e 疑似エコーのバッファにミを書き込みます。
f 疑似エコーのバッファにファを書き込みます。
g 疑似エコーのバッファにソを書き込みます。

Accidentals

疑似エコーバッファに書き込む音程の臨時記号です。
controlが音名の場合に、指定可能です。

+ …
♯ Sharp
# …
♯ Sharp
- …
♭ Flat
= …
♮ Natural
* …
♮ Natural

octave

疑似エコーバッファに書き込む音程のオクターブです。
controlが音名の場合に、指定可能です。
省略時は、現在のオクターブ（※ＭＭＬコンパイラ側が記憶している）になります。

内容（Contents）

疑似エコーのバッファを制御します。

注意（Notes）

※疑似エコーは、ＭＭＬコンパイラ側実装のコマンドです

疑似エコーバッファへの書き込みは、それを解消するために使います。

例（Sample）

例えば、こういう時に使えます。 


    v15 EC1,6,16                //疑似エコー設定
    EC-                         //疑似エコーバッファのリセット

    l4  o4  |:  cdef \ gfed :|	

    ECf                         //疑似エコーバッファに g を書き込み
                                //これがないと、 d の音でエコーを生成してしまう。

    gab`c
