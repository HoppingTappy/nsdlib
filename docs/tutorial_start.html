チュートリアル　準備

ファミコンで音楽を作るには、色々と用意が必要です。
ひとつづつ解説します。

nsd.lib

このソフトです。

Microsoft(R) Windows

はいってますよね？

テキストエディタ

お好きなテキストエディタをお使いください。
Windowsに付属のメモ帳でも良いのですが、作者は以下のエディタをお勧めします。

  • サクラエディタ
  • 秀丸エディタ（シェアウェア）

ＮＳＦプレイヤー

せっかく作った音楽データも、プレイヤーが無いと鳴りません。
プレイヤーをインストールしましょう。

VirtuaNSF

この、VirtuaNSFの最新版が一番お勧め。
元々はファミコン・エミュレータ`VirtuaNES'からサウンドの部分だけを取り出したもの。
なので、機能面としては一番再現性が高いです。

hoot

こちらもお勧め。
トラック毎に鍵盤表示がされる点、再生する音もファミコンに近いです。
