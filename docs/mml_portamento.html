MML reference: Portamento

定義（Define）

        Pdelay,rate,depth,target

引数（Inputs）

delay

ポルタメントがかかるまでの遅延時間です。
フレーム単位で指定してください。

rate

depthを加算する周期です。
フレーム単位で指定してください。

depth

rateフレーム毎に加算する値です。
16で100 centを意味します。

target

最終的な変化量です。1で半音に相当します。

内容（Contents）

次の音符に対して、ポルタメントをかけます。

注意（Notes）

depthとtargetの符号は合わせて下さい。

例（Sample）

    P20,1,-7,-24    c1  c1     /* Portamento */
