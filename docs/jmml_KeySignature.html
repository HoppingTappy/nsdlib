NSDL（NES Sound Driver Library）

トップ＼ 使い方＼ MMLコンパイラ＼ ストトン表記＼

ストトン表記：調

定義（Define）

        調{調指定}

引数（Inputs）

調指定

# …
以降に続く音名を半音上げます
＃ …
以降に続く音名を半音上げます
+ …
以降に続く音名を半音上げます
- …
以降に続く音名を半音下げます
♭ …
以降に続く音名を半音上げます
= …
以降に続く音名の調号を無しにします。
ラ …
A, La, ラ
シ …
H, Si, シ
ド …
C, Do, ド
レ …
D, Re, レ
ミ …
E, Mi, ミ
ファ …
F, Fa, ファ
ソ …
G, Sol, ソ

内容（Contents）

現在のトラックの調号を指定します。

注意（Notes）

ＭＭＬコンパイラーの疑似命令です。
次に調{}コマンドが来るまで、本コマンドで指定された音階に、自動で臨時記号が付加されてコンパイルされます。

各記述ブロック（音楽, 効果音）では、デフォルトは調号無しとなります。

例（Sample）

    調{#ファド}   レミファソラシ↑ドレ    /* ニ長調スケール */
    調{=ファド}   ドレミファソラシ↑ド    /* ハ長調スケール */
