NSDL（NES Sound Driver Library）

トップ＼ 使い方＼ MMLコンパイラ＼ ストトン表記＼

ストトン表記：リピートＢ

定義（Define）

        ｜：
	￥
	：｜

引数（Inputs）

無し

内容（Contents）

リピート（繰り返し）を記述します。
リピート(B)は、楽譜に記述されたリピートを記述することができます。

｜：
リピート(B)のカウンターをリセットします。
￥
1回目のリピート時は無視します。
2回目以降は:|コマンドで記憶したポインタにジャンプします。
：｜
直後のコマンドがある演奏ポインタを記憶し、無条件で|:コマンドの位置に戻ります。
１つの|:コマンドに対して１個以上記述できます。

注意（Notes）

リピート(B)の区間において、\コマンドは必ず1回記述してください。
リピート(B)のネストはできません。
リピート(B)の区間でリピート(A)を使用する場合、リピート(A)の区間を終了してからリピート(B)を終了してください。

例（Sample）

    |: c \ d :| e :| f :| g c1    /* = c d c e c f c g c1 */
